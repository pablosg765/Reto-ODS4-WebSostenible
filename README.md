# Reto-ODS4-WebSostenible

## 1. Análisis del Problema y Sostenibilidad

### El "Bug" Educativo

En muchos centros educativos todavía se utiliza una gran cantidad de papel para apuntes, tareas y materiales. Además, muchos estudiantes tienen dificultades para acceder a recursos educativos organizados en un solo lugar.

Esto provoca pérdida de información, falta de comunicación entre profesores y alumnos y un impacto ambiental negativo debido al uso excesivo de papel.

### Nuestro "Parche" (Impacto ODS 4)

Nuestra plataforma web permitirá compartir apuntes, tareas y materiales educativos de forma digital entre profesores y alumnos. Esto facilitará el acceso al aprendizaje desde cualquier lugar y mejorará la organización de los recursos educativos.

Además, la web estará diseñada de forma sostenible utilizando imágenes optimizadas y un diseño ligero para consumir pocos datos. También incluirá opciones de accesibilidad para que personas con discapacidad puedan utilizarla fácilmente.
